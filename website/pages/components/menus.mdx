## Styling

The `$selectable-color` SASS variable can be used to set the color range used
to indicate selectable items.  The default value is `"brand"`

You can also set the `--selectable-color` and `--selectable-background` CSS
custom properties to change the foreground and background colors respectively.
The default value for `--selectable-color` is `var(--brand-100)` and for
`--selectable-background` is `var(--brand-50)` when using a light theme and
`var(--brand-40)` for the dark theme.

The `--selected-color` is used for selected items. That defaults to
`--brand-40` for a light theme and `--brand-30` for a dark theme.
