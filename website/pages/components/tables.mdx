## Selectable Rows

Add the `selectable` class to make the rows appear selectable. This uses the
same colours as <MenusLink/> which can be changed using the
`$selectable-color` SASS variable or the `--selectable-color` and
`--selectable-background` CSS custom properties.

<Example
  html={Selectable}
  language="html"
  caption="Selectable Rows"
/>

## Vertical Alignment

The default vertical alignment for a table is `middle`.  Add the `top`,
`baseline`, `middle` or `bottom` classes to the `table` element or any `tr`,
`th` or `td` elements to change it. You can also set the `--valign` CSS custom
property to any of those values.
