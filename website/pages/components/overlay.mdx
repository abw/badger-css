# Overlay

Add the `overlay` class to an element to make it an overlay. The default
positioning is `absolute` so it will fill to cover the nearest parent that has
positioning set to `relative`.

<Split>
  <Snippet
    file="overlay/absolute.html"
    caption="Overlay"
  />
  <div className="output">
    <RelativeDemo/>
  </div>
</Split>

## Fixed Viewport

Add the `fixed` class to set the overlay position to `fixed` so that it covers
the entire viewport.

<div className="grid-2 gap-8 stack-desktop">
  <Snippet
    file="overlay/fixed.html"
    caption="Fixed"
  />
  <div className="output">
    <OverlayDemo/>
  </div>
</div>

## Light and Dark

The default overlay color is white for a light theme and black for a dark
theme.  You can explicitly add the `light` or `dark` CSS class if you want a
particular color overlay regardless of the theme.

{/*
  NOTE: we can't use Split here as that defines a container.  An element with
  position: fixed becomes fixed to the container not the viewport.
*/}
<div className="grid-2 gap-8 stack-desktop">
  <Snippet
    file="overlay/light.html"
    caption="Light Overlay"
  />
  <div className="output">
    <Overlay2 className="light" title="Show Light Overlay"/>
  </div>
  <Snippet
    file="overlay/dark.html"
    caption="Dark Overlay"
  />
  <div className="output">
    <Overlay2 className="dark" title="Show Dark Overlay"/>
  </div>
</div>
