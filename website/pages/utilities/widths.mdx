## Max Width Classes

The `max-width-Nrem` classes can be used to set the maximum
width of an element to a number of rem units.  In this case the width
of the element will be constrained to the width of the parent element.
You can also use the corresponding `max-width-Npx` classes
if you prefer to think in pixels.

The `width-*` and `max-width-*` classes work
by using the `--width` custom property which is set to the
appropriate number of rem units based on the `-Nrem` or `-Npx` suffix.
