# Fonts

## Font Classes

The `font-sans` CSS class can be used to select a sans-serif font,
`font-serif` for a serif font or `font-mono` for a monospaced font.

## Font Stacks

The default font stacks are designed to provide a consistent appearance across
different browsers and platforms.  They should usually match the operating
system default UI font and should be available in all font weights.  There are
also backup fonts included to render emojis.
