# Flex

## Flex Classes

Use the `flex` CSS class to set the `display` mode to `flex`.  The default
alignment mode (i.e. the `align-items` CSS property) is `stretch`.

<Example
  html={Flex}
  caption=".flex"
  expand
/>

## Gaps

The `gap-N` classes can be used to add a gap between successive elements.
Here `N` is a number from 1 to 16 representing multiples of the default
spacing unit (`0.25rem` by default).

<Example
  html={Gap}
  caption=".flex.gap-4"
  expand
/>

## Flex Evenly
