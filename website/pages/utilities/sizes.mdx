# Text Sizes

## Size Classes

Use the `smallest`, `smaller`, `small`, `medium` (default) `large`, `larger`
and `largest` CSS classes to set the text size. These are defined as SASS
variables which you can override (e.g. `$size-large`) and CSS custom
properties which you can use in your own stylesheets or inline styles
(e.g. `--size-large`).

## Size Multipliers

For larger text sizes you can use the size multiple classes: `x2`, `x3`, etc.,
up to `x10`. These are all multiples of `1rem`.
