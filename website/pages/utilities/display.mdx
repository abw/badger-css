# Display

## Display Modes

The `inline`, `block` and `inline-block` CSS classes can be used to set the
`display` mode.   These are useful when you have an element that would
normally have one display type that you want displayed in a different way.

See the <FlexLink/> and <GridLink/> pages for information about
using flex and grid display modes.
