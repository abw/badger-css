# Color Gradients

The `gradient` class can be added to an element to add a linear gradient
background.  The default colors are `color-40` and `color-60`, both of which
default to the corresponding shades of grey.  The default angle is 0&deg;
(i.e. bottom to top).

## Custom Gradients

The `--gradient-from` and `--gradient-to` CSS custom properties define the
start and end colors for the gradient. The `--gradient-angle` property sets
the gradient angle.
