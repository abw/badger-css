# Text Area Input

Text area inputs are much like <TextLink/> inputs.

One thing to note (that always catches me out) - if you want placeholder text
to appear then you have to make sure there is *nothing* (not even a newline)
between the opening and closing `textarea` tags.

## Inline Text Area Input

Text area inputs are full-width by default.  Add the `inline` class to give
