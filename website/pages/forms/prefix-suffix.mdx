focus rings work correctly, but they also adapt their colors for `valid` and

## Input Prefix

If you want to add a prefix to a field you should enclose the input in an
element with the `inputs` class and add the `prefix` class to the prefix
element.  You can add a prefix to a text input, textarea, select or a
checkbox or radio option with the `border` class set.  You can try it without
the `border` but the effect is lost somewhat.
