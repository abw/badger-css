# Fake Input

If you want something that isn't an input to look like an input then you can
add the `input` CSS class.

<Example
  html={Input}
  language="html"
  caption="Inline Text Input"
/>

## Inline Input

Just like regular inputs, they are full-width by default. Add the `inline`
class to give it a fluid width.

<Example
  html={Inline}
  language="html"
  caption="Inline Text Input"
/>

## No Hover

Add the `no-hover` class if you don't want it to change appearance on hover.

<Example
  html={NoHover}
  language="html"
  caption="No Hover"
/>

## Grid and Flex Layout

You can also use the <GridLink/> and <FlexLink/> classes to control layout.

  html={GridFlex}

## Disabled Inputs
