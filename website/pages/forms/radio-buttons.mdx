# Radio Buttons
## Radio Class

Radio buttons are implemented much like <CheckboxLink text="checkboxes"/>.
Create a `label` enclosing an `input` with the `type="radio"` attribute.

Badger-CSS identifies labels that include a radio button input using the
`:has()` CSS selector. All the major browsers currently support it, although
Firefox was a bit late to the party, only adding support in v121 released on
19th December 2023.

As a work-around to support older versions of Firefox, you can add the `radio`
class to the `label`.
